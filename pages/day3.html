Tecnologias estudiadas

  • React.js

    Es una biblioteca de JavaScript desarrollada por Facebook para construir interfaces de usuario interactivas y componentes reutilizables.

  • Vue.js

    Es un Framework progresivo de JavaScript que facilita la creación de interfaces dinámicas y es conocido por su facilidad de aprendizaje.

  • AngularJS

    Es un Framework desarrollado por Google que permite construir aplicaciones web de una sola página con un enfoque basado en componentes y TypeScript.

  • Svelte

    Es un Framework que compila los componentes en JavaScript puro, eliminando la necesidad de un runtime y mejorando el rendimiento.

  • TailwindCSS

    Es un Framework de CSS basado en clases de utilidad que permite diseñar interfaces rápidamente sin necesidad de escribir CSS personalizado.

  • C#

    Es un lenguaje de programación multipropósito desarrollado por Microsoft.

  • Java

    Es un lenguaje de programación de propósito general, concurrente y orientado a objetos.

  • Node.js

    Es un Entorno de ejecución de JavaScript basado en el motor V8 de Chrome, ideal para construir servidores rápidos y escalables.

  • MySQL

    Es un sistema de gestión de bases de datos relacional basado en SQL, ampliamente utilizado para almacenar y gestionar datos en aplicaciones web.

  • Spring Boot

    Es un Framework para Java que facilita el desarrollo de aplicaciones empresariales con configuraciones automáticas.
